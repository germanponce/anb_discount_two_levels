Multi Discount

Two Levels Discount on Sales and Purchases

Description

Shows a second level of sale/purchase discount reflected in accounting

This module shows a second level of discount in the sale order.

Moreover, it shows a second level of discount in the purchase order.

How it works, the calculation of the second discount is calculated as follows:

Sale price = 100€
First discount = 10% ---> 100€ x 0,1 = 10€ ---> 90€
Second discount = 25% of 90€ ---> 90€ x 0,25 = 22,5€ ---> 67,5€
Subtotal = 67,5€

Both discounts are transferred to the invoice and can be edited and created on the invoice itself.

Discounts are created and edited directly on the quotation and sale/purchase order lines.

Both discounts are printed in PDF Quote, Sale Order, Purchase Order and Invoice, and both are displayed in PDF if they are covered in form, otherwise not.

Installation

To install this module, you need to:
  • You must have installed or available to install the modules sale, account and purchase_discount.
You have the module purchase_discount available through this link: https://www.odoo.com/apps/modules/11.0/purchase_discount/

Usage

To use this module, you need to:
  • Check that your sales configuration in Odoo has checked Discounts.
